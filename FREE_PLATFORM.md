## Getting Started (3 Steps)

### Step 1: Install NeuralMesh (5 minutes)
```bash
git clone https://github.com/MrNova420/NeuralMesh.git
cd NeuralMesh
./deploy.sh
```

### Step 2: Add Your First Device (2 minutes)
```bash
# On your device (phone, Pi, laptop)
./neuralmesh-agent --server ws://your-main-server:3001
```

### Step 3: Deploy Something (1 minute)
- Open Template Gallery
- Click any template
- Click "Deploy"
- Done!
